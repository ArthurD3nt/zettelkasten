
> Libreria #js per gestire [[HTML]], [[DOM]], e CSS

## Installazione
```html
<head>  
<script  
	src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>  
</head>

```

- Tenere funzioni JQuery in un file separato

	<head>  
		<script src="my_jquery_functions.js"></script>  
	</head>
